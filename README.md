## To do App

### Stack

- TypeScript
- React
- Redux

### Goal

- 타입스크립트 적용
- 리덕스를 활용한 전역 상태 관리
- 투두리스트를 로컬스토리지에 저장해서 새로고침해도 유지하기
- 투두리스트 생성, 수정, 삭제하기
- 전체 투두 리스트 일괄 삭제하기
- 완료된 투두, 진행중 투두, 전체 투두 리스트 필터 버튼 만들기
